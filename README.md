# Wolf_3D.
The basic implementation of wolfenshtain 3D.

*Read this in other languages: [English](README.md), [Русский](README-ru.md).*

## Requirements

Mac OS El Capitan / Sierra / High Sierra

## Installation

```bash
git clone https://github.com/Arxcel/wolf_3D.git ~/wold_3D
cd ~/wold_3D
make
```
